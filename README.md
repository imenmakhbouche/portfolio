# Portfolio

## Description  
A personal portfolio website built with HTML, CSS, and JavaScript. Showcases projects and skills in a clean, user‑friendly layout.

## Technologies  
- HTML  
- CSS  
- JavaScript  

## How to Run  
```bash
git clone https://github.com/imenmakhbouche/portfolio.git
cd portfolio
# Open index.html in your browser
```

## Files  
- `index.html` — homepage  
- `style.css` — styling  
- `script.js` — interactive JavaScript (if any)  
- Additional assets (images, icons…)
